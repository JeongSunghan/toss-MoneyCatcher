# 머니 캐쳐 (Money Catcher) 🐱

2025년 토스 HTML5 with.넵튠 연습
네온 아케이드 스타일의 머니 캐칭 게임입니다. 떨어지는 돈을 휙득하여 명예와 부를 챙기세요.

## 🎮 게임 플레이

게임을 플레이하려면 다음 링크를 방문하세요:

**🎮 [머니 캐쳐 플레이하기](https://toos-money-catcher.vercel.app/)**

### 게임 방법
- **조작**: 좌우 스와이프 또는 방향키로 요원 이동
- **목표**: 돈을 모아서 부를 누리세요!
- **주의**: 세금(TAX)과 빚(DEBT)은 피하세요!
- **콤보**: 돈을 연속으로 받으면 콤보가 증가합니다

## 🚀 배포

게임은 Vercel을 통해 배포되어 있습니다:

**🔗 배포 URL**: [https://toos-money-catcher.vercel.app/](https://toos-money-catcher.vercel.app/)

### 로컬 실행 방법
1. `index.html`을 웹 브라우저에서 열기
2. "GAME START" 버튼 클릭
3. 게임 시작!

## 📁 파일 구조

```
MoneyCatcher/
├── index.html          # 메인 HTML 파일
├── game.js            # 게임 로직
├── style.css          # 스타일시트
├── assets/            # 게임 에셋
│   ├── money.png      # 돈 아이템
│   ├── point.png      # 포인트 아이템
│   ├── coupon.png     # 쿠폰 아이템
│   ├── tax.png        # 세금 아이템
│   ├── debt.png       # 빚 아이템
│   └── bg_night.png   # 배경 이미지
```

## ✨ 주요 기능

- ✅ 직장인 캐릭터 애니메이션
- ✅ 콤보 시스템
- ✅ 파티클 효과
- ✅ 반응형 디자인 (모바일/데스크톱)
- ✅ 터치/키보드 조작 지원
- ✅ 레벨 진행 시스템
- ✅ 아케이드 스타일 UI

## 🐛 수정 사항

### 버그 수정
- 레벨 진행 시 인덱스 범위 검사 추가

- 애니메이션 개선
